什么是Joggle？

Joggle 是一个款云端管理的内网穿透互联网应用。能够帮助用户快速的将内网应用映射到公网，帮助开发者实现穿透内网调试应用，网站程序映射到公网、映射RDP远程桌面链接家里PC电脑等等，能实现轻量级的网站应用映射。
实现这一切您只需要在您的电脑或者低功耗设备部署Joggle客户端即可。
