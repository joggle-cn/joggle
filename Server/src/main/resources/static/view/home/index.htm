什么是Bullet？

Bullet 是一个开放式内网穿透工具, 用于快速、便捷切换网络的路由器，它能帮助开发者实现内网调试，开发者与开发者之间的联调等工作。更能实现轻量级的网站应用，满足这些您只需要在您的电脑部署一个Bullet 客户端即可触手可及。

免费、开源

您可以通过github下载最新的源代码

灵活配置

Bullet Server提供强大的映射配置功能

多租户支持

采用Saas平台架构

多协议支持

支持TCP、HTTP、HTTPS等协议

Bullet 是完全开源的。它的代码托管、开发、维护都依赖 GitHub 平台。
